# ICU Neuro-Monitor 🏥
> AI-powered contactless ICU delirium detection using CCTV + audio — no wires, no patient contact.

🏆 **5th Place — YODHA Warriors of AI, 24-Hr National Hackathon | Jyothi Engineering College | Jan 2026**

---

## 🎯 What It Does
Transforms standard CCTV feeds and microphones into an intelligent delirium monitor for ICU patients.

- Real-time posture & fall risk detection
- Optical Flow for restlessness detection (even under blankets)
- Face blurring for patient privacy
- Noise (dB) & Light (Lux) environmental tracking
- Causality Engine — links environmental triggers to patient agitation
- Tiered audio alerts (Warning / Critical)
- Asystole Override — Flatline Mode if Heart Rate drops to 0

## 🛠️ Tech Stack
